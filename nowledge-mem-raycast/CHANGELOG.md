# Changelog

## [0.2.0] - 2026-02-14

### Added

- **Edit Working Memory**: inline edit form for `~/ai-now/memory.md`, saves directly
- **Copy actions everywhere**: Copy Content and Copy Title available on all memory items (list + detail)
- **Open in Editor**: open Working Memory in your default text editor

### Changed

- Store-ready: added keywords, publish script, LICENSE, assets/metadata structure

## [0.1.0] - 2026-02-14

### Added

- **Search Memories**: semantic search across your knowledge base with relevance scoring
- **Add Memory**: save insights with title, content, and importance from Raycast
- **Working Memory**: view your daily briefing directly in Raycast
- Deep link support: open memories in Nowledge Mem desktop app
- Copy-to-clipboard actions for quick pasting
